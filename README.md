# Calendar with Electron

## What can you do?
- Add Event
- Update Event
- Delete Event
- Import iCalendar (ics)
- Export as iCalendar (ics)

## Before launch

1. Install all dependency

```bash
  npm install
```

2. Generate Back-End files

```bash
  npm run build:back
```

3. Generate Front-End files

```bash
  npm run build:front
```

## Launch

```bash
  npm start
```
